# Code inventory

- `shuffle_control_core.py` is the primary implementation for data shuffling,
  full-data fitting, held-out fitting, validation, and report generation.
- `HMM_shuffle_full_fit.ipynb` and `HMM_shuffle_4fold_heldout.ipynb` are the
  executed notebooks.
- `build_notebooks.py` and `execute_notebook.py` are original notebook helper
  scripts retained as provenance artifacts.

The copied core has been adapted to the curated directory paths. The executed
notebooks retain their original outputs and are included to document the run.
